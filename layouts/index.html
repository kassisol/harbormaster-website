What is Harbormaster?

By default, Docker commands are run over Unix socket, but this can be changed by using TCP with TLS enabled. The latter allow certificates to be used to connect to Docker remotely. Certificate's CN (Common Name) is used as username for RBAC to control who is allowed to do what.

HBM

An application to authorize and manage whitelisted docker commands using Docker AuthZ plugin.

TSA

An application acting as a CA (Certification Authority) server to issue certificates for Docker Engines and users.

TWIC

A client application for HBM TSA to manage Docker certificates to connect to the Docker daemon using TLS.
