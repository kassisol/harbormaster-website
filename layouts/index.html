Harbormaster

Harbormaster is a set of tools to secure Docker environment.

Get Started
> mkdir /etc/systemd/system/docker.service.d
> vim /etc/systemd/system/docker.service.d/daemon.conf
> cat /etc/systemd/system/docker.service.d/daemon.conf
[Service]
ExecStart=
ExecStart=/usr/bin/dockerd --authorization-plugin=hbm
> systemctl daemon-reload
> systemctl restart docker.service
> curl https://github.com/harbourmaster/hbm
> hbm init
> hbm server
INFO [0000] Server has completed initialization
INFO [0000] HBM server         logdriver=standard storagedriver=sqlite version=0.2.0
INFO [0000] Listening on socket file
▋

Security

HBM extends the open source Docker Engine by adding authorization control.

Audit

All the operations are tracked.

Decoupled

Users don’t change their source code. Instead, admins write policies with an easy-to-use command.

Easy to Deploy

HBM has zero deployment dependencies. It runs as a daemon side-by-side with Docker Daemon.
